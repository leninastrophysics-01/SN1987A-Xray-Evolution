# SN1987A X-ray Evolution

## Project Overview
This project investigates the long-term X-ray evolution of Supernova 1987A using multi-epoch Chandra observations.

## Objectives
- Analyze soft and hard X-ray flux evolution
- Study hardness-ratio variations
- Investigate shock interaction with the circumstellar medium
- Examine transition to ejecta-dominated phase

## Data Source
- Chandra X-ray Observatory
- ACIS observations

## Methodology
- Data reduction using CIAO
- Spectral extraction
- Thermal plasma model fitting
- Flux evolution analysis
- Hardness-ratio analysis

## Key Findings
- Soft X-ray flux shows flattening after ring interaction
- Hard X-ray flux continues increasing
- SN1987A is transitioning toward an ejecta-dominated phase

## Tools Used
- CIAO
- Sherpa
- XSPEC
- Python
- Matplotlib
